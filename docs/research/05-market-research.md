## NOVEL FEATURE IDEAS (feasible with GEP events + OBS, offered by nobody)

1. **Round-scoped smart clips (context-bounded, not fixed-length).** Use GEP `round_start/round_phase/round_end` + kill burst detection to save replay-buffer segments bounded by *round semantics* â€” a 1v4 clutch exports as the whole round, a single kill as Â±8s. Every existing clipper (Medal, Outplayed, ShadowPlay) cuts fixed 15â€“60s windows; none are round-aware inside the streamer's real output. 2. **Dual-timeline marker fan-out ("one event, everywhere your editor looks").** A single GEP event simultaneously â†’ Twitch VOD marker (Helix), OBS recording chapter (30.2+ Hybrid MP4), and a CSV/EDL/FCPXML row â€” all with the *same* label ("R21 ace, Jett, Ascent") and correct per-sink timecode (stream time vs recording time vs file time). Solves the documented Premiere-ignores-chapters problem and the UserVoice "Export Markers" request in one step. No tool aligns even two of these sinks today.

3. **Zero-AI editor handoff pack on recording stop.** When recording stops, auto-emit: YouTube-paste-ready chapter block (match/map/ace granularity presets), highlight manifest JSON (event type, score context, weapon, timestamp), and per-event thumbnails from the recording. Replaces $10â€“20/mo post-hoc AI chapter/clipping SaaS with deterministic, instant, local output â€” a "your editor opens one folder" workflow nobody sells.

4. **Anti-stream-snipe scene automation.** GEP phase events (agent select, map load, queue) auto-switch OBS to a "lobby-safe" scene and back at round start â€” protecting ranked streamers from snipers without a moderator or Stream Deck reflexes. Only ever hand-rolled per-game (cf. [apex-legends-obs-game](https://github.com/kratosator/apex-legends-obs-game) scene switcher); no multi-game, event-exact product exists.

5. **Native event-driven overlay source with live state machine (kill streak / clutch / KDA).** An OBS source (not a browser hack) rendering session stats and 1vX clutch state computed from roster + kill events in real time, themeable, working offline with zero CV/GPU cost â€” directly attacking Game Pulse's two weaknesses: Streamlabs-Desktop exclusivity and AI-vision overhead. Bonus novel sub-feature: viewer-visible "clip meter" that fills as multikills accumulate, gamifying the auto-clipper for chat.
