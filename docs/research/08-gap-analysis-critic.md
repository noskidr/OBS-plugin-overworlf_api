## 3. Risks nobody flagged

- **R1 â€” ow-electron ToS/distribution (biggest omission).** Overwolf Developer Terms: "If your Application is powered by Electron, you must enter into the additional terms set out in Exhibit B" (legal.overwolf.com/docs/overwolf/developers/developer-terms/). Production requires DevRel proposal approval + Developer Console QA review; runtime packages (gep/overlay/recorder) are closed-source and fetched from Overwolf's servers at runtime â€” Overwolf holds a de-facto kill switch over the companion app, and first run requires internet/CDN access. Third-party distribution is permitted ("through the Overwolf Platform and through a third-party distributor") but at developer's sole liability; ad-policy/CMP obligations attach if monetized (rev share: 15% subs / 30% ads under the main Terms).
- **R2 â€” Twitch marker failure modes.** Bucket is 800 points/min per client-per-user (ample), but the guide warns "Some API endpoints may return HTTP 429 response codes for reasons unrelated to the general rate limit bucket" â€” undocumented per-endpoint throttles possible on kill-spam. Markers hard-fail if VOD storage is off â€” which is the **default** for non-affiliate accounts â€” so the flagship feature silently 404s for a large user cohort without a preflight check; 140-char description cap; marker position vs event latency unmeasured.
- **R3 â€” Event forgery via unauthenticated localhost WS.** Browsers allow `ws://127.0.0.1` connections from arbitrary web pages: without a pairing token, any website a streamer visits could inject fake events â€” saving replays, spamming Twitch markers, polluting recordings. Needs an auth handshake and origin checks from day one.
- **R4 â€” GEP operational fragility.** Game patches break events regularly (the status endpoint exists for this); GEP module versions (`min_gep_version_electron`) update out-of-band from Overwolf servers; Valorant coverage exists only while Riot tolerates Overwolf/Vanguard whitelisting. The system needs a health-check + degrade-gracefully path, not just happy-path forwarding.
- **R5 â€” Competitive timing.** Streamlabs Game Pulse/AI Highlighter list OBS support as "planned" and Twitch Auto Clips is in alpha â€” the empty-slot advantage the market report celebrates has a shelf life; ship the thin core first.
